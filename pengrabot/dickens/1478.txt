Once you have a region selected, the most basic commands are: If you are running Emacs in its own window (and not within an  "xterm" window) as part of an X Window session, you can select  the region simply by using the mouse.  Click the mouse's left button at  the start of the area to be selected, and drag the mouse to the end of the region. The area you selected should be highlighted. You cannot cut text directly with the mouse in the X Window System. Version: trunk
