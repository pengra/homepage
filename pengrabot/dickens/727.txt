Practice-it is a web application to help you practice solving Java programming 			problems online. Many of the problems come from the University of Washington's 			introductory Java courses. To use Practice-it, first create an account, then choose a problem from our 			list. Type a solution and submit it to our server. The system will test it and 			tell you whether your solution is correct.
